## 기술 스택
### Front-end
- React (Vite)
- Tailwind CSS
### Back-end
- Flask
- Python
### AI/OCR
- CNN 기반 커스텀 모델
- EasyOCR (오픈소스)
### Tools
- IDE : PyCharm, VSCode
- Version Controll : Git, Github
